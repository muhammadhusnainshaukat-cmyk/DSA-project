# Ride-Sharing Dispatch & Trip Management System

A DSA-based in-memory ride-sharing system inspired by Uber/Careem.

## Features
- Graph-based city routing
- Driver & rider management
- Trip lifecycle state machine
- Rollback mechanism
- Tkinter GUI

## Tech Stack
- Python
- Tkinter
